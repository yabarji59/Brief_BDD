1 - REQUETE POUR MODIFIER LA BDD

Exemple : modifier nom de famille d'un membre

UPDATE membres 
SET nom = Nouveaunom
WHERE num_carte = 202122458;




2 - REQUETE POUR SUPPRIMER LIGNE DE BDD 

DELETE FROM editeurs where id_editeur = 1;
=> ne marche pas si l'option ON DELETE de foreign key de livres est restreint. Il faudrait dans notre cas faire un 
ALTER TABLE livres DROP FOREIGN KEY FK_livres_relation__editeurs;
puis un 
ALTER TABLE livres 
ADD CONSTRAINT fk_livres_relation_editeurs FOREIGN KEY (id_editeur) references editeurs(id_editeur) ON DELETE SET NULL;

On peut aussi mettre CASCADE mais ça supprime alors les livres qui viennent de cet éditeur là et ainsi de suite selon les foreign keys des livres concernés.

Les seules lignes de BDD qu'on peut supprimer dans notre base sont celles de la table prêts. 
DELETE FROM prets where id_pret = 1; par exemple.
